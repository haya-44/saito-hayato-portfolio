使用可能なツール・言語

HTML / CSS

様々なデザイン制作がある中でweb制作にも興味を持ち、知見を深めるために独学をしております。
HTML/CSSの「基本的な記述方法」部分的にサイトの編集を加えたりすることができます。

Photoshop

「Photoshop」を用いてjpg画像の白抜き、合成・加工ができモックアップ画像の制作も可能です。
「Illustrator」で書き出し「Photoshop」で立体的な表現、ぼかしを加えたりなど、細かい表現の時に使っており、デザイン制作においてクオリティを上げることができます。

Illustrator

Illustratorを用いてチラシ、パンフレット等のDTP制作が可能です。
カーニング、禁則処理、行間を意識したレイアウトの調整ができ、印刷物の基本的な取り扱いも問題なくできます。
一番使い慣れているソフトです。
